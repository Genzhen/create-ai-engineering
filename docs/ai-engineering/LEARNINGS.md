# 经验积累

> 记录开发过程中积累的经验、最佳实践和踩坑记录。

---

## 使用说明

这个文件用于记录团队在开发过程中积累的经验：

- **踩坑记录**：遇到的 Bug 和解决方案
- **最佳实践**：发现的好做法
- **性能优化**：优化经验和结果
- **安全经验**：安全相关的注意事项

## 添加记录

每次解决重要问题后，AI 会提示是否添加到 LEARNINGS.md。

---

## 记录模板

```
### [日期] [主题]

**问题**：[描述遇到的问题]

**原因**：[分析根本原因]

**解决**：[描述解决方案]

**经验**：[总结经验教训]
```
